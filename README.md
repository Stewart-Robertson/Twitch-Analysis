# Twitch-Analysis

This project uses 
* Python
* Pandas
* SQL
* Matplotlib & Seaborn
* GeoPy and Nominatim
* Folium
to analyse data collected from Twitch Insights.

## NOTE
To view the interactive folium map created at the end of the notebook, copy the url of the notebook into the search bar of https://nbviewer.jupyter.org/
